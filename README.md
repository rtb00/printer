# PDF Binder Formatter

Transform your PDF into a format where printed pages can be stacked, allowing you to selectively remove any unwanted pages before binding them into a book seamlessly.

## Overview

This project takes your PDF document and reformats it so that you can:
- Print in a layout optimized for folding and assembly.
- Remove unwanted pages easily.
- Bind the remaining pages into a professional-looking book.

## Features

- **Page Reordering:** Rearranges pages for efficient double-sided printing.
- **Custom Page Removal:** Exclude pages that you do not want to include.
- **Easy Binding Layout:** Prepares your PDF for a flawless book-binding process.
- **User-Friendly Interface:** Features a Tkinter-based GUI â€” simply start the program and interact through the graphical menu.
- **Cross-Platform:** Works reliably on most operating systems.

## Installation

Ensure you have Python 3.7+ installed. Then, clone the repository and install the required dependencies:

```bash
git clone https://github.com/rtb00/pdf-binder-formatter.git
cd pdf-binder-formatter
pip install -r requirements.txt
```

## Usage

### Starting the Application

Since this project now features a Tkinter graphical interface, you only need to start the program to use it. Launch the application by running the following command:

```bash
python pdf_binder.py
```

This will open the GUI, where you can load your PDF and select pages to remove. No command-line arguments are necessary.
